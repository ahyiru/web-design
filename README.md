### 运行

可自行配置运行项目，如运行 `web-design` 项目：

```js
npm start ---dirname=app

```

默认运行 `app` 目录。`npm start`

[low-code](./low-code.md)

***demo项目***

```js
npm start ---dirname=demo

```
demo包含mock数据。

***template项目***

```js
npm start ---dirname=template

```
